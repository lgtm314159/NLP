Jet

Jet, the Java Extraction Toolkit, provides a set of tools for natural language analysis and specifically to support information extraction.  The tools in the toolkit can be used in three ways:
  • through tool control scripts, which support the simple, unconditional application of a series of tools to selected portions of a message
  • in some cases, as separate, stand-alone tools -- 'Jet Utilities'
  • through the Java API, which allows the combination of the tools with maximum flexibility
